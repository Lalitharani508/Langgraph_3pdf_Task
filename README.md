# 🤖 AI Knowledge Assistant (RAG + LangGraph)

## 📌 Overview
This project is a Retrieval-Augmented Generation (RAG) application built using LangGraph, OpenAI, and Streamlit. It answers questions based on PDF documents.

## 🚀 Features
- PDF-based question answering
- LangGraph workflow
- Vector database (Chroma)
- Streamlit UI

## 🛠 Tech Stack
- LangGraph
- LangChain
- OpenAI
- ChromaDB
- Streamlit
